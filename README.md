# Credit Card Management System

A Python console-based program to manage credit cards with the following features:

## Features
- Add new credit cards with username, password, card number, and limit
- Show all credit cards in a readable format
- Spend from a credit card while updating available limit and outstanding amount
- Pay bills to reduce outstanding amount and restore available limit
- Stores data in a JSON file with username/password validation for security

## Technologies Used
- Python
- JSON
- File Handling

## How to Run
1. Clone the repository:  
   ```
   git clone https://github.com/shivani63/Credit-Card-Management-Python.git
   ```
2. Navigate to the project folder:  
   ```
   cd "credit card"
   ```
3. Run the program:  
   ```
   python main.py
   ```

## Example Usage
```
1: Add credit card
2: Show credit cards
3: Spend
4: Pay bill
5: Exit
```
- When spending or paying, the program will ask for **username, password, card number, and amount**.
